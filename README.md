# 24V-Relay

This repository contains the Altium Designer files created for the schematic and PCB design for Triton AI's latest 24 V relay. This 6-channel relay uses mosfets to switch a 24 V input and relay it to an output. As of April 28, 2026, the board has 3 channels being used to switch on the go-kart's 24 V LED which signals whether the go-kart is in emergency mode (red), semi-autonomous mode (yellow), or autonomous mode (green).
